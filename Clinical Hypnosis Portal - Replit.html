    I have created the vanilla version of the application for you in a new folder called vanilla-version. This version uses standard HTML, CSS, and plain JavaScript, making it even easier to copy to your PC and modify without needing a complex build system like React.

    All the core logic is now combined into a few focused files within that folder:

    • server.js: The complete backend (Node.js/Express) including the database schema and API routes in one place.
    • index.html & admin.html: The frontend pages.
    • styles.css: All the styling for both pages.
    • script.js & admin-script.js: The frontend logic for the main site and the admin area.

    You can find all these files in the vanilla-version directory in your workspace. This structure is perfect for local development on your PC—just run node server.js (after setting your database connection) and you're good to go!
